# Git Integration

There are numerous plugins to integrate git with Vim.

## View Git Changes

Inline git changes,

```vim
Plug 'airblade/vim-gitgutter'
```

## Git Blame History

```vim
Plug 'rhysd/git-messenger.vim'
```

## View Git History

```vim
Plug 'cohama/agit.vim'
```
